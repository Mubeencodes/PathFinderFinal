# React + TypeScript + Tailwind Frontend

## Quick start

1. Copy `.env.example` to `.env` and set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` .
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start server:
   ```bash
   npm run dev
   ```

The server runs on `http://localhost:8080` by default.
